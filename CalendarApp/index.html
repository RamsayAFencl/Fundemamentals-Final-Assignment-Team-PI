Team PI's Live Calendar App

Days with events are the color of the most recently added event

The current day will be in bold

Selecting an Event

Click on a date(s) to add to selection

Selected date(s) will glow their designated color

Click the "clear selected" to clear the selection

Adding an Event

Click on a date(s) to add to selection

Click the "add event" button to add an event to the selection

Enter in the event's information on the pop-up

Viewing an Event

Click on a date(s) to add to selection

Selected day(s) event information will be displayed below the calendar

Deleting an Event

Click on a date(s) to add to selection

Selected day(s) event information will be displayed below the calendar

Click the "delete" button for the event you want to delete

Confirm the deletion on the pop-up

Note that events can only be deleted one at a time

The Calendar will store your data inside local storage, so it will keep all data until the browsers memory is cleared
